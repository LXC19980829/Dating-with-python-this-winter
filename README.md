#  Dating-with-python-this-winter

这是一个 Python 基础课程。

## 前言

课程目的是让大家了解编程—— 最好是编程思想，掌握 Python 的基本操作。考虑到大家来自各种各样的专业，爬虫指定是不行，对大家没啥用，客户端或者后端也是对程序员才有用。所以，一个具体目的暂定是，学会 Python 基础之后，学数据分析 —— 掌握 Numpy, Pandas 等框架。


## 课程结构

**课前问题**：大家可以试着完成。

**课中讲解和实践**：四十多分钟的直播。

**课后编程实践**：自己动手实践作业。

隔一天上一节课，一天空闲是留给课前问题和课后编程的。
